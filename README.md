# MEERLEBEN

Premium-Landingpage für den Full-Service-Auswanderungs- und Immobilienbegleitservice MEERLEBEN – vom Leben in Deutschland zum neuen Zuhause am Meer in Spanien.

## Technologie

- Next.js mit App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP ScrollTrigger
- Responsive und barrierearm

## Start

```bash
pnpm install
pnpm dev
```

Danach die Seite unter `http://localhost:3000` öffnen.

## Produktion

```bash
pnpm build
pnpm start
```

Animationen werden bei aktiviertem `prefers-reduced-motion` automatisch reduziert.

## Deployment

Das Repository ist für Vercel und Node.js-Hosting bei Hostinger vorbereitet. Die vollständige Anleitung steht in [DEPLOYMENT.md](./DEPLOYMENT.md).
